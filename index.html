About

소개

QA 없이 배포하는 1인 개발자와 초기 팀을 위한 AI 품질 모니터링 서비스 AIM을 혼자 만들어 자체 도메인에서 운영하고 있습니다. 설계부터 브라우저 자동화, LLM 통합, CI 연동, 단일 VM 운영까지 서비스 수명주기 전체를 완주하는 것을 좋아합니다.

팀에서는 RAG 파이프라인과 객체 탐지 모델을 직접 다뤘고, 결과는 항상 평가 지표로 증명하려 합니다. 이 페이지의 모든 숫자는 실측입니다.

  • 가용성 · SSL · Lighthouse · Playwright 시나리오를 한 번에 실행하는 통합 검사 파이프라인을 설계하고 구현했습니다 (FastAPI + Celery + PostgreSQL).

  • 시작 3주 차에 자체 도메인 HTTPS 실서비스에 도달했습니다 — 커밋 269 · 머지 PR 115 · 테스트 460개(백엔드 337 + 웹 123, 2026.07 실측).
  • CI 배포 훅과 Slack/Discord webhook 알림을 연동하고, AIM 자신을 첫 프로젝트로 등록해 60분 주기로 스스로를 모니터링합니다.
